HTML CheatSheet

Tag Name Description
<span> The Span Tag It's an inline container for text and can be used to group text for styling purposes.
<strong> The Strong Tag highlights important, serious, or urgent text and browsers will normally render
this highlighted text in bold by default.
<a> The anchor element The <a> anchor element is used to create hyperlinks in an HTML document.
The href='' determines the location the anchor element points to.
<br> The Line Break Element The line break element will create a line break in text and is especially useful where a division of text is required.
